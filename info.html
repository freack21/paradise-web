Informasi

  • Hypertext Markup Language
  • Bahasa dasar dalam pengembangan web dan merupakan pondasi dari hampir semua halaman web
  • Membuat struktur dan mengatur konten di halaman web
  • HTML bekerja dengan menggunakan elemen : CSS (Cascading Style Sheet), JavaScript
  • Cascading Style Sheet (CSS) adalah bahasa style sheet yang digunakan untuk mengatur tampilan dan format dari sebuah dokumen yang ditulis dengan bahasa markup
  • CSS digunakan untuk mengubah tampilan (style) dari halaman web.
  • Terdapat 2 istilah penting, yaitu bahasa style sheet dan bahasa markup.
  • Bahasa Style Sheet merupakan format bahasa khusus yang terdiri dari kumpulan kode untuk mengatur tampilan dari sebuah dokumen.
  • Bahasa Markup merujuk kepada dokumen yang menggunakan “tanda” atau “mark”.
